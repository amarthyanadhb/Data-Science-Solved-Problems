# Data-Science-Solved-Problems

This repository contains 20 solved data science and machine learning problems.
Each problem includes a separate dataset, question, and fully implemented solution.

## Topics Covered
- Python & Data Analysis
- Exploratory Data Analysis (EDA)
- Statistics & Hypothesis Testing
- Regression Models (MLR, Logistic Regression)
- Tree & Ensemble Models
- Dimensionality Reduction (PCA)
- Clustering Techniques
- Time Series Analysis
- Recommendation Systems
- NLP & Neural Networks
- Text Classification using RNN

## Technologies Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn, Statsmodels
- TensorFlow / Keras (for NN & RNN)

## Repository Structure
Each problem folder contains:
- Dataset
- Problem statement
- Jupyter Notebook solution
- Explanation and conclusions
